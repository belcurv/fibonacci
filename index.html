Each svg <rect> element's width and height are always the same and must be positive numbers. The positioning x,y coordinates can be negative. Wrapping the <rect> and <path> elements in a <g>roup allows for easy rotate transforms. For example: transform: rotate(90deg).

The x and y positioning coordinates are funky. What's the relationship? Assuming counter-clockwise assembly:
Xn = Math.abs(Yn-1)
Yn = -( Widthn-2 + Yn-2 )
